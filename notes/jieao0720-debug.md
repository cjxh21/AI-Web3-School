# 钱包

## 内部关系

助记词是基础，由一个助记词可以导出多个私钥，而公钥和私钥是一对一的，通过哈希化把私钥转化为公钥，公钥用来做交易的签名；  
同时，所有的资产（币）都是存在区块链上面，并不是在钱包上，因此不同的钱包可以等上同一个账号（通过助记词、私钥）；  

# 交易

## 交易构成

要做的事、Gas Fee、Nouce、Signature；  
其中，Gas Fee用来激励区块的提议者和质押的用户；Nouce是交易的序号，避免重复交易的发生；Signature即签名，必须要通过私钥签名才能提交交易。

## 智能合约

智能合约本质上为代码，规定了交易和区块的底层逻辑，任何链上活动都以智能合约为执行基础；

智能合约之于社会，它显著降低了信任成本，规范了交易的流程，并且透明、可查看，也是区块链的核心价值之一。
