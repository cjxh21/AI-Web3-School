Claude Code 给我制定的第一天的学习计划

**Day 1：账户、地址与钱包的本质**

  **学习目标：** 理解私钥 → 公钥 → 地址的推导关系，明白钱包只是私钥管理工具。

  **核心概念：**

  - 私钥：256 位随机数，是链上身份的唯一凭证

  - 助记词：私钥的人类可读编码（BIP-39），一组助记词可派生多个私钥

  - 地址：公钥经哈希后的标识符，公开可见

  - 钱包：管理私钥、构造并签名交易的客户端工具

  **阅读材料：**

  - Ethereum Accounts 官方文档

  - MetaMask Getting Started 指南

  1. 安装 MetaMask，创建一个测试钱包

  2. 用文字说明地址、助记词、私钥分别是什么

  3. 记录：为什么助记词/私钥绝对不能泄露、截图、上传
