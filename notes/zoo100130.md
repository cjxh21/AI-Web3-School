# 今天將Hemers安裝到外接硬盤上

首先在外接硬盤上裝上WSL

然後配置環境

```
wsl -d Hermes-Ubuntu-22.04
```

安裝並驗證

```
hermes --version
```

並安裝好Node v22.22.3、npm 10.9.8、ripgrep、ffmpeg，把 hermes 固定到 /usr/local/bin/hermes。我也關掉了這個 WSL 的 Windows PATH 繼承，避免 Windows 路徑裡的括號干擾 Linux 指令。  
  
下一步配置API Key  

```
wsl -d Hermes-Ubuntu-22.04
```

然後在WSL裡  

```
hermes setup
```

最後啟動

```
hermes
```
