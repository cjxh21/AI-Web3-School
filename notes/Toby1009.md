# 開營＆Hermes 安裝

當了助教，認識了好多新的小夥伴

也裝上了 Hermes，發現其實安裝挺簡單的，就是不知道跟 Claude 或 Codex 的遠程差異多大？

理論上可能是要參考：[https://github.com/nousresearch/hermes-agent](https://github.com/nousresearch/hermes-agent)

但是我是直接問 Claude 怎麼裝 Hermes，然後就給了我 CLI Command 很方便

`curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash`

裝完後：

```
hermes setup    # 初始化設定（選擇模型、API key 等）
hermes doctor   # 確認環境是否正常
hermes          # 啟動 agent
```

後續也挺方便的吧，就會跳出要不要配置 TG，要選什麼模型之類的

我是配了 TG，模型用 Codex ChatGPT 5.5

然後再玩了一下，使用我常見的調研方式，用 Chrome-mcp 去搜尋 X，和 Obsidian 整理筆記，調研了我自己
